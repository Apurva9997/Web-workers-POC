Web Workers Demo

This demonstration shows the difference between processing data in the main thread vs using web workers. Try sorting the large array with both methods and observe how the UI responsiveness changes.

When using the main thread for heavy computation, the UI becomes blocked and unresponsive. Web Workers allow intensive processing to run in a background thread, keeping the UI responsive. This demo uses bubble sort to sort a list of 10,000 numbers to demonstrate the difference.
